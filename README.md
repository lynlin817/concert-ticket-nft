# Concert Ticket NFT Prototype

This repository contains a simple ERC-721 smart contract prototype for NFT-based concert ticketing.

## Features
- Mint a unique concert ticket NFT
- Store a metadata URI
- Check ownership with ownerOf
- Track ticket usage with used
- Mark a ticket as used after entry
